## Para rodar localmente
### ctrl+c ctrl+v no terminal

```bash
git clone https://github.com/DanielZubcov/AraraBotSite.git
cd AraraBotSite
npm install
npm run dev
```

### Para adicionar melhorias ou mudanças, e preciso após o git add, e commit utilizar o comando

```bash
npm run build
ou
npm run deploy
```
